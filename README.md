# 🎤 豐原向陽店 KTV 分帳計算機

一個純前端的單頁工具，依「人數、日期類型、進場時間、歡唱時數」自動算出**每人應付金額**，並支援**企業合作折扣**。資料來自豐原向陽店消費方式計價表。

## 功能

- 下拉選擇人數（最多 30 人），自動帶入對應包廂（可手動更改）
- 四種日期類型：平日（一~四）、星期五、星期六、星期日，各自的時段折數
- 依**進場時間**判定時段；**跨時段按分鐘計算價差**
- 「買二送 N」自動折減計費時數（每 2+N 小時只收 2 小時；鐘點計價則全收）
- 顯示**離場時間**與**歡唱總時數**
- 企業合作折扣（目前含「台中美光 85 折」），只套用於包廂費、且在時段折扣之後
- 即時顯示總計與每人金額、完整費用明細

## 計費規則

```
包廂費 = Σ(每分鐘包廂原價 × 該分鐘所在時段折數) × 企業折扣   （僅計費時段，免費時段不計）
清潔費 = 清潔原價 × 清潔折數   （08:00~16:59 五折、其餘原價、星期日全天原價；企業折扣不打）
服務費 = (包廂原價 × 歡唱時數 + 清潔原價) × 10%   （以折扣前原價計算；企業折扣不打）
總計   = 包廂費 + 清潔費 + 服務費
每人   = 總計 ÷ 實際人數
```

> 跨時段、現場活動及實際收費以門市現場為準，本工具僅供估算參考。

## 發布到 GitHub Pages

1. 建立一個新的 GitHub repository，把 `index.html` 推上去。
2. 進入 repo 的 **Settings → Pages**。
3. **Source** 選 `Deploy from a branch`，分支選 `main`、資料夾選 `/ (root)`，儲存。
4. 稍候即可透過 `https://<你的帳號>.github.io/<repo 名稱>/` 開啟使用。

本工具為單一 `index.html`，無任何相依套件，也可直接用瀏覽器開啟本機檔案使用。

## 自訂企業折扣

編輯 `index.html` 中的 `COMPANIES` 陣列即可新增公司：

```js
const COMPANIES = [
  { name: "無（一般價）", rate: 1 },
  { name: "台中美光（85 折）", rate: 0.85 },
  // { name: "其他公司（9 折）", rate: 0.9 },
];
```
